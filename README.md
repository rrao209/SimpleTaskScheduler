# SimpleTaskScheduler
Task scheduler to schedule jobs on a single Node

### Documentation
- [Setup](#setup)
- [Installation](#installation)
- [run](#run)

# Setup
```
git clone https://github.com/rrao209/SimpleTaskScheduler.git
cd SimpleTaskScheduler

```

# Installation
mvn clean install

# Run
mvn exec:java
